## Topic: The Art of Cleaning Data
Today, I focused on the most critical step in Machine Learning: **Data Preprocessing**. I learned that raw data is rarely perfect and needs cleaning before training a model.

## ðŸ§  Key Concepts Learned:
1. **Handling Missing Values:**
   - **Deletion:** Removing rows with too much missing data.
   - **Imputation:** Filling missing values with Mean (Average), Median, or Mode using `fillna()`.
2. **Removing Duplicates:**
   - Identifying and dropping repeated rows to prevent bias.
3. **Handling Outliers:**
   - Detecting extreme values (e.g., Age = 200) using visualization (Boxplots) and removing them.
4. **Data Transformation:**
   - Standardizing formats (e.g., date formats, lowercasing text).
